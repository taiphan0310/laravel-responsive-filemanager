## Steps to contribute
1. Fork [taiphan0310/laravel-responsive-filemanager](https://github.com/taiphan0310/laravel-responsive-filemanager.git) from GitHub.
1. Run commands below:

    ```
    git clone git@github.com:taiphan0310/laravel-responsive-filemanager.git
    cd laravel-responsive-filemanager
    make init
    ```
1. Edit codes in `vendor/taiphan0310/laravel-responsive-filemanager`
1. Push your changes to your fork.
1. Send a pull request to [taiphan0310/laravel-responsive-filemanager](https://github.com/taiphan0310/laravel-responsive-filemanager.git).
